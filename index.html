My Web Development Project

Fronapp

Fronapp is a web application develop with PHP for backend, HTML,CSS and Javascript for the frontend. Fronapp is my first strong personnal project : it's my baby ! In brief, it's an application which allows the users to listen Webradios for free. It's necessary to create a account to use this application. After you're logged in, it's the same behavior as Spotify. The only difference is that Spotify plays music while Fronapp only plays radio. It's a copyright issue, of course. Same as Spotify, Fronapp allows his users to add songs in a favourite folder. At this day, I pursue my development on this project and in the future, I'm going to try to monetize this application to pay for hosting charges.

Concerning people's testimonials towards me on this project, I just have a user feedback since I'm the only one to develop this application. They talk about a very successful and useful project with a fast reaction time when it comes to fixing a bug. It's very pleasant when you know that I spent a lot of time to do it. This project which is very close to my heart will be detailed with more precision in the VLOG.

Bee Challenge
Bee Challenge is a project that consists in helping people discover the botanical garden of Nancy and its plants. The application contains a map with all the zones of the garden to be discovered with the plants present in each zone as well as their properties for an educational purpose. You just have to scan the barcodes next to the plants to get all the information about the plant. The objective is to scan as many plants as possible to have the most points in the given time. It's a group project and the grade given by the teacher will be the only testimony of the project.
Media Library
This project is a media library realized only in frontend (HTML, CSS, JS). It is a list of different media such as movies, albums and video games. The main goal was to be able to create any media with the possibility to modify it whenever you want. Through a search bar, it is also possible to add a movie by its name with all its caracteristics, using a external database. With a local storage system, there is a persistence of the data which allows to keep everything that has been added or modified. The project had to be carried out alone in order to check our knowledge on the subject. To this day, I am still waiting for the professor's testimony to give me an idea of the level of success of what I have done.
